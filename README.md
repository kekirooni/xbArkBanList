this is outdated and hasnt been maintained since 2019/2020, this is only being posted for preservation/educational purposes. (the authentication method is currently broken)

The original functionality of this script, was to read the ArkAPI banlist endpoint and display the recent bans on the Xbox Official Network on Ark Survival Evolved, 

utilising the xbox API framework to get the Gamertag from the XUID.

By the end of its development course, I had scraped enough API endpoints from the Xbox PC application to do basically anything; which leads to the super cool "Add to Group" functionality.

The selected XUID would be sent an invite to a group named CHEATERS GET BANNED, and the script would send a message of the Globally Banned messagebox popup that you get ingame upon joining an official server,

but in the groupchat. 

After this was finished I developed another script called xbArkAutoChat, which the entire functionality of this was scanning the banlist and inviting new additions into the group all automatically, i will also

be uploading that too.
